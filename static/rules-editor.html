LLM Gateway - Configuration Editor (v1.11)

Export configurations for coding agents

Download the models configuration in the format required by OpenCode or GitHub Copilot.

Include fallback provider models (e.g., OpenRouter models)

OpenCode integration allows you to use the LLM Gateway models directly in your OpenCode environment.

  1. Download the configuration JSON file below.
  2. Copy the content and merge it into your ~/.config/opencode/opencode.jsonc configuration file.
  3. For more detailed instructions, please refer to the official documentation: OpenCode Models Documentation.

GitHub Copilot integration enables you to use LLM Gateway models as custom language models in VS Code.

  1. Download the configuration JSON file below.
  2. Copy the content and merge it into your chatLanguageModels.json configuration file.
  3. For more detailed instructions, please refer to the official documentation: VS Code Agent Customization.
